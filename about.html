Pihak-Pihak yang ikut :

Font Awesome

Font Awesome adalah sebuah librari yang biasanya dipakai untuk membuat icon. librari ini Freemium. ada fitur gratis tapi dibatasi untuk pemgunaan iconnya. tetapi versi gratisnya sudahlah sangat lengkap

Google Font

Google Font adalah librari font gratis. Google Font meliki banya sekali Font. kelibihanya adalah librari ini memiliki CDN, jadi tidak usah repot untuk mendownload. tinggal pasang link dan pakai

Netlify

Netlify adalah layanan Cloud Computing. Netlify menawarkan layanan sebagai hosting gratis tanpa server untuk aplikasi web atau situs web statis.

Github

Github adalah tempat berbagi proyek pengembangan perangkat lunak. Github menawarkan repositori untuk penyimpanan kode sumber perangkat lunak
